# Paymaster RCP Service

This is a project built using FastAPI, Uvicorn, and JSON-RPC Server.

## Installation

1. Run `python3 -m venv .venv` to create a virtual environment
2. Run `source .venv/bin/activate` to activate the virtual environment
3. Run `make install` to install the dependencies
4. Run `make run` to run the server

## Usage

The server will be running at `http://localhost:8000`.

### API Endpoints

- `/`: Returns a JSON response with a message "Hello, world!"
- `/docs`: Interactive API documentation generated by Swagger UI
- `/redoc`: Interactive API documentation generated by ReDoc

### JSON-RPC Endpoints

- `/rpc`: JSON-RPC endpoint for performing mathematical and string operations

## Testing

1. Install the testing dependencies using `pip install -r requirements.txt`
2. Run the tests using `pytest`
